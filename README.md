# The Shadow Files

Real horror stories. Real people. Real places. The Shadow Files is a true scary story podcast for listeners who like the lights off and the door locked. Every episode is a single unsettling first-person account, 12 to 15 minutes. New episodes Sunday, Tuesday and Thursday.
